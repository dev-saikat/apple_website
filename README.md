## Setup Instructions
1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/apple-website.git
    ```
2. Navigate to the project directory:
    ```bash
    cd apple-website
    ```
3. Open `index.html` in your browser to view the website.
